
---

## 🛠 Features

- Home page with product listing and search functionality  
- Product details page  
- Shopping cart functionality using JavaScript  
- Responsive design for desktop and mobile  
- Interactive UI components (buttons, hover effects, etc.)  

---

## 💻 How to Run

1. Open the `index.html` file in your browser  
2. Navigate through the homepage, add items to cart, and explore features  
3. No server setup required, fully front-end project  

---

## ⚙️ Requirements

- Modern web browser (Chrome, Firefox, Edge, etc.)  
- Text editor or IDE (VS Code recommended)
